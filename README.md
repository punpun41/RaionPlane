## 🎮 About the Project

**[R-Ship]** is an action-packed 2D top-down shooter built in the Unity Engine. Players pilot a fighter plane, navigating through waves of enemies while managing their health and dodging incoming fire. This project was created to demonstrate core game development mechanics, including object pooling, physics-based movement, and UI management.

### ✨ Key Features
* **Dynamic Player Movement:** Smooth, responsive controls for navigating the plane.
* **Shooting Mechanics:** Continuous firing system to take down enemy aircraft.
* **Health & UI System:** Interactive health bars and on-screen UI to track player status.
